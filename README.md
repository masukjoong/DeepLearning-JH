# DeepLearning-JH

## Jupyter notebook

### install

```shell
pip3 install jupyter
```

### shortcut option

- jupyter notebook location
  
  - `%localappdata%\Programs\Python\Python310\Scripts\jupyter-notebook`

- shortcut option
  
  - `--notebook-dir=<location>`
  
  - ex) `--notebook-dir=D:\workspace\DeepLearning-JH`
